QUEM SÃO OS GURI?

Cobblemon - Os Guri é um servidor de Minecraft focado no mod Cobblemon e modpack Cobbleverse, que traz a experiência completa de Pokémon para dentro do Minecraft.

Aqui você pode explorar biomas específicos pra encontrar Pokémon selvagens, construir sua base, batalhar com outros jogadores e se tornar o melhor treinador do servidor!

As dúvidas mais comuns

O servidor é gratuito? ▼
Sim! O Cobblemon - Os Guri é 100% gratuito para jogar. Você só precisa ter o Minecraft Java Edition original e baixar nosso modpack pelo Google Drive, também sem custo nenhum.
Precisa de uma versão específica do Minecraft? ▼
Sim! O servidor roda na versão 1.21.1 do Minecraft Java Edition. Você não conseguirá conectar com outras versões. Certifique-se de usar o launcher correto configurado com a versão certa pelo modpack.
Como eu adquiro o primeiro pokemon no servidor? ▼
Você precione a tecla C para adquirir o primeiro pokemon inicial.
